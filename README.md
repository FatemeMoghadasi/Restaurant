# Restaurant
A simple CLI-based restaurant management system using Python and PostgreSQL.
The system allows the manager to:
- Manage menu items (add, edit, view)
- Manage tables (add, remove, view status)
- Manage orders (create, update status, view details)
- View daily sales report
## Prerequisites
- Python 3.x
- PostgreSQL installed and running
- psycopg2 library (`pip install psycopg2`)
